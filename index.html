Olá, eu sou davi muniz

Estudante de desenvolvimento web apaixonado por criar soluções inovadoras e aprender novas tecnologias. Durante este semestre, mergulhei profundamente no mundo do desenvolvimento front-end, explorando conceitos fundamentais e técnicas avançadas.

HTML5 CSS3 JavaScript Tailwind CSS Responsive Design

Expectativas vs Realidade

Expectativas Iniciais

Acreditava que desenvolver interfaces seria mais sobre código do que design. Imaginei dominar React rapidamente sem uma base sólida em JavaScript.

Realidade Descoberta

Percebi que UX/UI é crucial. Entendi a importância de fundamentos sólidos antes de frameworks. A progressão orgânica trouxe entendimentos mais profundos.
